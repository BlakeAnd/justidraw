# justidraw

Experimental DAW.

This is made to work with a drawing tablet on Windows (wintab driver).


# controls

### drawing
* left click: draw
* hold ctrl: erase
* middle mouse: pan
* hold ctrl: zoom

### shortcuts
* ctrl+Z: undo
* ctrl+Y / ctrl+shift+Z: redo
* ctrl+S: save 
* ctrl+O: open save folder 

Drag and drop save files to open them!
Your last save file will be loaded on startup.

* Escape: quit
* Delete: clear all
